2. 데이터 삭제 방법

방법 A. 일부 데이터만 삭제

  1. 앱에서 삭제할 항목(운동 루틴, 운동 기록, 체중 기록, 인바디 기록, 회복 기록, 사진, 사용자 정의 운동 등)으로 이동합니다.
  2. 해당 항목을 개별 삭제합니다.
  3. 삭제는 즉시 기기에 반영됩니다.

방법 B. 모든 데이터 삭제

  1. 앱 안에서 설정 → 모든 데이터 초기화를 누릅니다. 모든 루틴·운동기록·체중·인바디·회복 기록·사진이 한 번에 영구 삭제됩니다(되돌릴 수 없음). 또는
  2. 기기의 설정 → 애플리케이션 → 오토핏 → 저장공간으로 이동해 데이터 삭제(저장공간 지우기)를 누릅니다. 또는
  3. 앱을 삭제(제거)합니다. 앱을 삭제하면 기기에 저장된 모든 오토핏 데이터가 함께 제거됩니다.

방법 C. 백업 파일 삭제

백업 내보내기로 만든 파일은 사용자가 선택한 저장 위치에 보관됩니다. 더 이상 필요하지 않으면 해당 파일을 직접 삭제하세요.

앱을 사용할 수 없는 등 도움이 필요하면 richanseon@gmail.com 으로 문의해 주세요.
